# Accessibility Audit Tool

## Overview
This project is a monorepo foundation for an accessibility audit application focused on WCAG insights, maintainable architecture, and accessibility-first development.

## Tech Stack
- Frontend: Angular, TypeScript, Tailwind CSS, Angular CDK
- Backend: Node.js, Express, TypeScript
- Shared: TypeScript contracts for API payloads

## Project Structure
- frontend/: Angular application
- backend/: Express API
- shared/: shared TypeScript types

## Local Development Setup
1. Install dependencies with npm install.
2. Start the backend with npm run dev --workspace backend.
3. Start the frontend with npm run start --workspace frontend.
